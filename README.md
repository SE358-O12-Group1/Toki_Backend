## 5. Getting Started

To get started with the project, follow these steps:

1. Clone repository:

```
git clone https://github.com/plongtran239/ecommerce-nodejs-backend
```

2. Run project:

-   Install packages

```
npm install
```

-   Run project with local database

```
npm run dev
```

Or run project with cloud database

```
npm run dev:pro
```
